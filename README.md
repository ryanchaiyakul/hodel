## HoDEL

### Getting Started

1. Install [uv](https://docs.astral.sh/uv/getting-started/installation/) package manager.
2. In the root folder of this repository, execute `uv sync`.

> If you have CUDA device, `uv sync --extra cuda` installs `jax[cuda13]`.
